Exploding lines into lines

This tutorial teach you how to paste a block of text into individual lines. Gif animation display at the end. Suppose you have: // Comment1 // Comment2 // Comment3 To be exploded into each line of below: foo = 1; bar = 2; baz = 3; As this form: foo = 1; //Comment1 bar = 2; //Comment2 baz = 3; //Comment3 Vscode and Sublime can copy a block then use mutil-line editing shortcut to write to multiple lines.
